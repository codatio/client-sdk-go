# WriteStatus

The current status of the write request, which is the same as the push operation status.


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `WriteStatusPending`  | Pending               |
| `WriteStatusFailed`   | Failed                |
| `WriteStatusSuccess`  | Success               |
| `WriteStatusTimedOut` | TimedOut              |
